# RAG Browse

Browse past sessions and events: "$ARGUMENTS"

- No argument → use `mcp__plugin_claude-rag_rag__browse_sessions` to list recent sessions
- UUID argument → use `mcp__plugin_claude-rag_rag__browse_sessions` with session_id for details
- "projects" → use `mcp__plugin_claude-rag_rag__browse_projects` to list all projects

Offer next actions: drill down into sessions, or search within results.
